WikiSwitcher

(2024)

WikiSwitcher est une extension Google Chrome qui permet de choisir automatiquement la meilleure langue pour un article donné de Wikipédia parmi les langues selectionnées par l'utilisateur.
java-script chrome-extension

Book-Indexing

(2023 - 2024)

Outil permettant de construire un index de recherche d'un livre basé sur l'ensemble du texte jusqu'à un point précis. La motivation est de pouvoir interroger ce qui a déjà été lu pour rafraîchir le contexte concernant les personnages et les événements survenus précédemment dans le livre sans gâcher les événements au-delà du point choisi.
julia python javascript text-analysis
